# 指标分析
投机性太高，交易量少的股票不适用

## KDJ 超买信号
当k线数值高于90，同时D线数值高于80以上，并且J线数值连续三天高于100以上时，即称之为KDJ超买，提示股价已进入超买区，短期回调概率较大。持股者应暂时考虑减仓规避风险，空仓者应继续空仓观望，避免盲目追高。

## KDJ超卖信号
当k线数值低于10以下，同时D线数值低于20以下，并且J线数值连续三天低于0以下时，即称之为KDJ超卖，提示投资者股价已进入超卖区，股价短期下跌动能减弱，反弹概率较大，持股者可以继续持股等待反弹，短线空仓者也可择机介入，博取股价反弹差价。


## 公式
```python
KDJ(N=9, M1=3, M2=3)
RSV = (CLOSE - LLV(LOW, N)) / (HHV(HIGH, N) - LLV(LOW, N)) * 100
K = EMA(RSV, (M1 * 2 - 1))
D = EMA(K, (M2 * 2 - 1))
J = K * 3 - D * 2
```

- LLV n天最低价
- HHV n天最高价
- RSV 当日收盘价格在n天区间的位置，如果收盘价的曲线上升即可表示是上升趋势，比如rsv=80 表示是最近n天区间价的80%
- EMA 权重平均值 EMA(n,10) =  n*C/n! + (n-1)*C[1]/n! + ...,与MA的区别是越靠近当天的权重越高，即加权平均值
- MA  平均值 = (c +c[1]+...c[n])/n
- K 以M1*2-1天为周期计算收盘价趋势的平均值
- D 以K值再次计算EMA平均值可以放缓K值特征，变的更平均
  - 当K值高于D值证明是上升通道（这段时间都是平均上升的，如果偏离幅度越高），反之下降通道。当KD交叉证明趋势反转
- J 证明KD关系的背离趋势

## 策略分析

### 推论
#### 买入原则
1. J的斜率变大第2天高概率盘中价格会高于前一天的收盘价(即 J-J[1] > J[1]-J[2])
2. J的值比前一天高买入获利的概率高
3. J的斜率变大时量也放大，证明有交易量支撑上涨
4. 是否进入超买区间的结果验证
5. 是否进入超卖区间进行买入更合理
6. K大于D值

#### 算法待补全
1. 需要参考ma20或30后，看是否能提升成功率

#### 不买入原则
1. K小于D时J的斜率变高也不买
2. 收盘价在5日均线以下不买
3. K值高于80买入和K值低于20买入的概率
